Bordes

Los bordes de un elemento de HTML se definen con tres atributos básicos: estilo, anchura y color.

  1. border-style: se refiere a la forma del borde y puede tomar estos valores: none, hidden, dotted, dashed, solid, double, groove, ridge, inset, outset.

    Ejemplo:

    
            p {
                border-style: dotted;
            }

    Se mostraría:

    Lorem ipsum dolor sit amet, consectetur adipisicing elit. Minima esse dolor odio eveniet nesciunt. Voluptatibus, reiciendis sit. Error, recusandae, repellendus nam vero numquam in minima hic quos unde, dicta ullam.

  2. border-width: El ancho del borde puede tomar valores numéricos absolutos y relativos (px, mm, em,...) y los valores predeterminados: thin, medium, thick.

    Ejemplo:

    
            p {
                border-width: medium;
            }

    Se mostraría:

    Lorem ipsum dolor sit amet, consectetur adipisicing elit. Minima esse dolor odio eveniet nesciunt. Voluptatibus, reiciendis sit. Error, recusandae, repellendus nam vero numquam in minima hic quos unde, dicta ullam.

  3. border-radius: define el radio de círculo de las cuatro esquinas.

    Ejemplo:

    
            p {
                border-radius: 10px;
            }

    Se mostraría:

    Lorem ipsum dolor sit amet, consectetur adipisicing elit. Minima esse dolor odio eveniet nesciunt. Voluptatibus, reiciendis sit. Error, recusandae, repellendus nam vero numquam in minima hic quos unde, dicta ullam.

  4. border-color: Con esta propiedad las esquinas de los bordes se pueden hacer redondeadas. El valor está compuesto por dos valores el radio horizontal y el radio vertical. Si sólo se da un valor la esquina tendrá forma redondeada, y si se dan dos valores forma de elipse.

    Ejemplo:

    
            p {
                border-color: darkviolet;
            }

    Se mostraría:

    Lorem ipsum dolor sit amet, consectetur adipisicing elit. Minima esse dolor odio eveniet nesciunt. Voluptatibus, reiciendis sit. Error, recusandae, repellendus nam vero numquam in minima hic quos unde, dicta ullam.
