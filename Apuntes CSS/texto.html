Textos, tipografías y tamaños

El texto es fundamental en el diseño de una página web. La fuente o tipo de letra o tipografía es la forma de dibujar los caracteres de un alfabeto. Existen multitud de fuentes para cada alfabeto o sistema de escritura, no hay un método universal de denominación y clasificación de las fuentes. CSS proporciona uns sistema para seleccionar y usar fuentes basado en las "familias de fuentes".

font-family

La propiedad font-family indica la fuente a emplear. La fuente se indica como una lista de valores separados por comas. El navegador intentará utilizar la primera fuente y si no es posible, intentará con las siguientes. Se puede dar un valor de fuente de dos formas: Familia de fuentes y fuentes genéricas.

- Familia de fuentes, se indica el nombre concreto de la fuente, como "Times New Roman", Arial, Verdana, etc. Si el nombre tiene espacios, irá entre comillas.
- Fuente genérica se indica un conjunto de fuentes similares, por ejemplo, sans-serif, es un grupo de fuentes sin "remates", pertenecen a este tipo Arial, Verdana, Helvetica, ...

Fuentes web

Como hemos dicho las fuentes deben estar disponibles en el momento de utilizarlas. En el dispositivo que utilicemos (ordenador, tablet, móvil, etc.) ya están instaladas las fuentes más comunes. Si en nuestra página utilizamos una fuente nueva o poco común debemos asegurarnos que esté disponible para todos los usuarios que visiten la página. Podemos lograrlo de dos formas, utilizando unas fuentes web o instalando la fuente en nuestro servidor web.

Para ello debemos visitar la web https://fonts.google.com/ y seguir las instrucciones. Básicamente se trata de elegir una fuente de las disponibles y seguir unos de estos dos métodos:

La forma más usual de hacerlo es utilizando las fuentes web de Google.

- Colocar un enlace a la web de Google indicando la fuente elegida, se hace desde la cabecera de la página HTML. 


                <link href="https://fonts.googleapis.com/css?family=Aldrich" rel="stylesheet">

- Importar la fuente en nuestra hoja de estilos desde la web de Google. 


                <style> @import url('https://fonts.googleapis.com/css?family=Aldrich'); </style>

font-style

La propiedad font-style permite indicar si la fuente es inclinada. Puede tener tres valores:

  1. normal, no inclinada. Es el valor por defecto.
  2. oblique, la forma de la letra no cambia, simplemente se inclina.
  3. italic, además de inclinarse, la forma de algunas letras cambia

font-size

La propiedad font-size establece el tamaño de la fuente. El valor se puede indicar en varias unidades de medida, unidades absolutas (px, mm, ...) unidades relativas (em, %, ...) o con valores predeterminados (xx-small, x-small, small, medium, large, x-large y xx-large).

font-weight

La propiedad font-weight establece el grosor de la fuente. El valor se puede indicar con valores predeterminados (normal, bold, lighter y bolder) y con estos 9 números: 100, 200, 300, 400, 500, 600, 700, 800, 900.

text-align

La propiedad text-align define la alineación horizontal de un texto. Se pueden aplicar los siguientes valores:

  1. left, texto alineado a la izquierda.
  2. right, texto alineado a la derecha.
  3. center, texto alineado al centro de su contenedor, dejando igual espacio a la derecha que a la izquierda en cada línea.
  4. justify, texto justificado, es decir, el texto se alinea a la derecha y a la izquierda en cada línea, para ello se introducen espacios en blanco entre las palabras.
    5. 
                text-align: justify;

text-shadow

La propiedad text-shadow agrega sombra al texto. Acepta una lista de sombras separadas por coma, para aplicarlas al texto y a la propiedad text-decorations del elemento.
