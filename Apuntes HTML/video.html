Vídeo

La etiqueta vídeo incrusta un vídeo en una página web, dentro de la etiqueta se debe especificar la fuente del vídeo mediante la etiqueta source, por ejemplo: 


                        <video  controls>
                            <source src="multimedia/video.mp4" type="video/mp4">
                                Formato de vídeo no soportado.
                        </video>

El atributo controls hace que aparezca una barra con los controles para arrancar/parar la reproducción, control del volumen, etc.

Antes del cierre de la etiqueta </video> se puede escribir un texto que aparecerá si el navegador no es capaz de reproducir el formato de vídeo. Aunque algunos navegadores no lo muestran y muestran un mensaje propio como "Origen no válido".

Atributos

  1. autoplay. Si aparece este atributo el vídeo comenzará a reproducirse automáticamente al cargarse la página, si no aparece el usuario tendrá que pulsar en el botón Play.
  2. loop. Si aparece este atributo cuando el vídeo llegue al final volverá a reproducirse desde el principio, y así indefinidamente.
  3. muted. Si aparece este atributo no se oirá el sonido. Si utilizamos este atributo es conveniente incluir el atributo controls para que el usuario pueda activar el sonido.
  4. poster. Este atributo permite colocar una imagen como portada del vídeo
  5. width y height. Indican la anchura y altura del vídeo en pixels.
  6. preload. Indica cómo se debe cargar el vídeo, tiene estos tres valores: auto, none y metadata
  7. type. Este atributo dentro de la etiqueta source indica el tipo de vídeo. Los tipos mp4, ogg, webm, son los tipos reconocidos oficialmente por html 5.

Audio

Para reproducir sonido disponemos de la etiqueta audio. En su versión mínima sólo necesita una fuente de sonido y un control de reproducción. 


                        <audio controls src="multimedia/audio_prueba.mp3" >
                            Audio no soportado
                            </audio>

Esta etiqueta audio es muy similar a la que acabamos de ver para vídeo, si no estamos seguros de que el navegador vaya a poder reproducir el formato de vídeo podemos incluir etiquetas source con otros formatos, también podemos incluir el atributo type con el tipo de formato de audio.También disponemos de los atributos autoplay, controls, mute, loop y preload.

El tipo de vídeo mpeg (.mp3) es claramente el más utilizado en la web y es reproducido por todos los navegadores, otros tipos estándar de HTML 5 son ogg y wav.

Existen otros tipos de audio como el .m4a que utiliza la grabadora de voz de Windows.
